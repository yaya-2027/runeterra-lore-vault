# runeterra-lore-vault

A fan lore project for League of Legends / Runeterra, written in Obsidian.  
This repo also includes **VaultGraph** — a local graph visualizer to explore note connections, inspired by Obsidian's graph view.

> ⚠️ Fan work — see [DISCLAIMER.md](./DISCLAIMER.md)

---

## 📁 What's inside

- `/League of Legand` — Obsidian vault with lore notes (official + personal interpretations + original characters)
- `/app` — VaultGraph, the graph visualizer *(coming soon)*
